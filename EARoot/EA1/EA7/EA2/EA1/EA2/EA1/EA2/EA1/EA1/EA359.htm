System Initialization Function  Function: void SYS_Initialize( void *data )  Summary: Function that initializes all modules in the system.  Description: This function initializes all modules in the system, including any drivers, services, middleware, and applications.  Precondition: None.  Parameters: data            - Pointer to the data structure containing any data necessary to initialize the module. This pointer may be null if no data is required and default initialization is to be used.  Returns: None.  Example:<br/>#lt;code#gt; SYS_Initialize ( NULL );  while ( true ) { SYS_Tasks ( ); }<br/>#lt;/code#gt;  Remarks: This function will only be called once, after system reset.<br/>
